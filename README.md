# آيات وأذكار الرقية والتحصين — Ayat v2

A professional Islamic web app presenting Ruqyah and protection verses with dynamic audio playback.

## Features

- 🌙 5 sections of Ruqyah & protection verses
- 🎧 Dynamic audio via Quran API (Mishary Al-Afasy)
- 📱 Fully responsive — mobile + desktop
- ⌨️ Keyboard navigation (↑ ↓ arrow keys)
- 🎨 Dark mode with emerald & gold accents
- 🕌 RTL Arabic typography (Amiri font)

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- [Quran API](https://alquran.cloud/api) for audio
- Deployed on Vercel

## Usage

Open `index.html` in any modern browser, or visit the live deployment.
